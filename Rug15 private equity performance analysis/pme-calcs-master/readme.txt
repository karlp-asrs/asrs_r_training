pme-calcs contains files for demonstrating private equity performance calculations

the main r code is in 'pme functions.r'
'pme.rmd' is a markdown file for an RPubs post on the topic
'pedata.csv' is hypothetical data to help illustrate the calculations

This code and data was used for a presentation for R Finance 2014
'pe_performance_r_finance_2014.pdf' is the conference slide deck
'pe performance r finance 2014.lyx' generated the slide deck
